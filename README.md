# Expense Tracker

**Автор:** Васильченко Данил Владимирович

## Описание программы
Expense Tracker — это GUI-приложение на Python (Tkinter) для учёта личных расходов. Программа позволяет добавлять расходы, отображать их в таблице, фильтровать записи по категории и диапазону дат, а также считать общую сумму расходов за выбранный период.

## Возможности
- Добавление расхода с полями: сумма, категория, дата.
- Проверка корректности ввода:
  - сумма должна быть положительным числом;
  - дата должна быть введена в формате `ГГГГ-ММ-ДД`.
- Просмотр расходов в таблице.
- Фильтрация:
  - по категории;
  - по дате (от / до).
- Подсчёт общей суммы расходов за период.
- Сохранение данных в JSON.
- Загрузка данных из JSON.
- Удаление выбранной записи.

## Как использовать
1. Установить Python 3.
2. Скачать проект или клонировать Git-репозиторий.
3. Перейти в папку проекта.
4. Запустить файл:
   ```bash
   python expense_tracker.py
   ```
5. Ввести сумму, выбрать категорию и дату.
6. Нажать кнопку **«Добавить расход»**.
7. Для фильтрации указать категорию и/или диапазон дат, затем нажать **«Применить фильтр»**.
8. Для подсчёта суммы ввести период и нажать **«Подсчитать сумму»**.
9. Для сохранения или загрузки данных использовать кнопки **«Сохранить JSON»** и **«Загрузить JSON»**.

## Примеры тестов
### Тест 1. Корректное добавление
- Сумма: `450.50`
- Категория: `Еда`
- Дата: `2026-04-30`
- Ожидаемый результат: запись появляется в таблице.

### Тест 2. Некорректная сумма
- Сумма: `-100`
- Ожидаемый результат: выводится сообщение об ошибке.

### Тест 3. Некорректная дата
- Дата: `30-04-2026`
- Ожидаемый результат: выводится сообщение об ошибке формата даты.

### Тест 4. Фильтрация по категории
- Категория фильтра: `Транспорт`
- Ожидаемый результат: в таблице остаются только расходы категории `Транспорт`.

### Тест 5. Подсчёт за период
- Период: `2026-04-01` — `2026-04-30`
- Ожидаемый результат: отображается сумма всех расходов за апрель.
